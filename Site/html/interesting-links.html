    Links para os programas alfanuméricos do Arduino

  1. Software do Arduino Programming Language

    O software Arduino (IDE) de código aberto facilita a escrita de código e o upload para a placa. Este software pode ser usado com qualquer placa Arduino.

  2. Repositório do Arduino no Github

    Este repositório contém apenas o código para o próprio IDE do Arduino. Originalmente, ele também continha o núcleo e as bibliotecas do Arduino AVR e SAM (ou seja, o código que é compilado como parte de um esboço e executado no dispositivo Arduino real), mas eles foram movidos para seus próprios repositórios. Eles ainda são baixados automaticamente como parte do processo de compilação e incluídos nas versões compiladas.

    Links interessantes

  1. Arduino Brasil - Tutorial Tinkercad

    Playlist de tutoriais de tinkercad. Do canal Arduino Brasil, ensinando passo a passo ferramentas e funções de arduinos. O canal também conta com uma variedade de conteúdos, não se limitando apenas ao Tinkercad.

  2. Site oficial de arduino - página inicial.

    Site oficial de arduino. Conta com explicação de software, hardware, documentação e exemplos built-in. Existem partes voltadas à educação para o uso de arduinos e tem uma aba exclusiva para as redes sociais do site, com fóruns e comunidades.

  3. O que é Arduino: Para que serve, vantagens e como utilizar

    Blog falando dos benefícios do arduino e como funciona. A página de arduinos fala da história dele, como surgiu, usos, exemplos de usos, etc. O site também conta com várias outras áreas da computação e mostra ter vários tutoriais e aulas pagas de diversos temas da área, como robótica e modelagem 3D.

  4. An Experimental Comparison of Arduino IDE Compatible Platforms for Digital Control and Data Acquisition Applications.

    O estudo realizou uma pesquisa experimental para comparar o tempo de execução de ações por dispositivos incorporados que podem ser programados com a IDE do Arduino como operações matemáticas básicas e execução do código de programação.

  5. Arduino II: Systems; Steven F. Berrett.

    O documento contém uma breve revisão sobre os conceitos básicos do Arduino UNO R3 e sobre sua plataforma de desenvolvimento. Além disso contempla e discute sobre o ambiente de desenvolvimento Arduino (de modo aprofundado) e sobre o hardware do dispositivo.

  6. Arduino as a learning tool.

    O artigo introduz brevemente o hardware e o software do dispositivo Arduino e avalia o Arduino como um dispositivo de aprendizagem. Além disso demonstra algumas aplicações do Arduino como método de aprendizado.

  7. Podcast de arduino.

    Podcast falando a respeito do arduino, suas funções e como utilizar. Entre os assuntos discutidos, tem-se informações sobre onde comprar arduinos, como começar a usar, quais as áreas no mercado que será utilizado, entre outros. O episódio conta com 1h30 de duração.

  8. Top 10 2019-2020 arduino projects.

    Site que mostra diversos projetos feitos em arduinos em meados de 2020, incluindo um projeto de leitura de digital, um carro de controle remoto com sensor de movimento de mão wireless e um mecanismo que avisa motoristas quando eles estão distraídos enquanto dirigem por muito tempo.

  9. Vendas de arduino. FlipFlop.

    Loja virtual de placas e componentes para projetos envolvendo arduino, raspberry pi, impressora 3D, entre outros envolvendo robótica. Dentre os acessórios disponibilizados para arduino temos: kit de arduino para iniciantes, kit intermediário e avançado. Todos eles incluem resistências, cabos de conecção, protoboard, um arduino e alguns tipos de sensores.

  10. GitHub - arduino/Arduino: open-source electronics platform

    Página do GitHub com o código da IDE do arduino disponibilizado para download. O repositório está linkado com o site https://arduino.cc e também apresenta algumas informações sobre a instalação e a segurança do código.
